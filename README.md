# 梦境电台 v12 产品逻辑强化版

本版本在 v11 多页面版基础上完成 5 项改进：

1. 首页新增产品路径流程条：选梦境类型 → 定制偏好 → 生成内容 → 睡前播放 → 会员/礼盒。
2. 顶部导航精简为：首页 / 创作中心 / 睡前播放 / 会员 / 礼盒。
3. 播放页新增沉浸睡眠模式，降低睡前干扰。
4. 生成后新增“AI 生成结果确认页”，展示摘要、章节、适配理由和操作入口。
5. 会员与礼盒嵌入用户路径：生成结果页会自然推荐会员能力和关联 IP 礼盒。

## 运行

后端：

```bash
cd backend
npm install
npm run dev
```

前端：

```bash
cd frontend
npm install
npm run dev
```

浏览器打开：

```text
http://localhost:5173
```

后端默认：

```text
http://localhost:8787
```


## v16 修正说明

本版本修复了 v15 中“把带 UI 的视觉稿当作背景图”的问题。
现在第二页与播放页的动态背景全部替换为纯场景图：

- 末世小说：废弃温室 / 地下花园氛围背景
- 十二星座视频：星光图书馆与天文观测站背景
- 睡前故事：雨夜书店背景
- 白噪音声景：月夜窗边温馨角落背景

页面上的卡片、播放器、按钮和表单仍然由真实 React 组件渲染，不再嵌在背景图里。

## v22 修改说明

根据电子商务课程展示需求，本版本完成 5 项修正：
1. 顶部导航恢复“内容商店”和“数据看板”，购物车保留在右上角。
2. 首页定位改回“AI 个性化睡前内容电商平台”。
3. 首页新增明显的“内容商店入口卡片”，突出 6.9 / 9.9 / 12.9 元内容包。
4. App 展示区压缩高度与文案，不再抢占电商模块位置。
5. 最终打包时删除 node_modules、dist、.DS_Store、__MACOSX，避免依赖污染和运行错误。


## v23 新增功能：输入设定生成完整故事

本版本在“末世小说”和“睡前故事”定制页新增：
- 用户自定义设定输入框
- 必须出现元素
- 不要出现元素
- 狗血/爽文程度
- 根据设定生成完整故事或脚本
- 结果页展示完整故事预览
- 播放页同步显示用户设定

示例：女主重生回末世前三天，觉醒空间异能，囤货建安全屋，前男友和白月光后来跪求收留。
